📋 Формат файла Excel / CSV

  1. Первая строка — заголовки колонок
  2. Обязательные: ФИО умершего (или фио, умерший)
  3. Дополнительные: Родственник, Телефон, Адрес, Регион, Отношение
  4. Форматы: .xlsx, .xls, .csv
📂

Нажми чтобы выбрать файл Excel или CSV

📝 Формат текстового файла

  1. Каждый клиент начинается с: <b>ФИО</b> ДД.ММ.ГГГГ
  2. Адрес умершего: Адрес цели: ...
  3. Родственники: #1 ФИО ДД.ММ.ГГГГ
  4. Данные родственника: Адрес:, Паспорт:, СНИЛС:
  5. Телефоны: после строки Номера: — каждый с новой строки через • или -
  6. Разделитель между клиентами: пустая строка или ---
